In Sprint 3, the focus is on creating high-fidelity UI designs for the Digital Talent Management Platform. In this sprint, high-fidelity screens are being designed for the Employee, Manager, and Admin modules, including dashboard layouts, profile management, and approval workflows. Additionally, a design system has been created that includes color styles, typography, and reusable UI components to ensure consistency across the platform. These designs will serve as the final visual reference for the platform before moving to the development stage.
